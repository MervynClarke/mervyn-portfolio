# mervynclarke.com — Personal Portfolio

> *Gong Fu Cha (功夫茶) — "Making tea with great skill."*

A personal portfolio built with **Next.js 14**, **Tailwind CSS**, and **Framer Motion**.

## Quick Start

```bash
# Install dependencies
npm install

# Run dev server
npm run dev

# Build for production
npm run build && npm start
```

## Design Philosophy

Every section is an "infusion" — revealing a new layer of depth, inspired by the Chinese tea ceremony Mervyn studied and led at Penn State.

## Tech Stack

| Layer       | Technology              |
|-------------|-------------------------|
| Framework   | Next.js 14 (App Router) |
| Styling     | Tailwind CSS            |
| Animations  | Framer Motion           |
| Fonts       | Playfair Display, Inter, JetBrains Mono |
| Hosting     | Vercel (recommended)    |

## Project Structure

```
src/
├── app/          # Next.js App Router pages & layout
├── components/   # All UI components
├── hooks/        # Custom React hooks
└── lib/          # Data & utilities
```
